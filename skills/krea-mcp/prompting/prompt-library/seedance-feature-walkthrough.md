# Feature Walkthrough — Seedance

**Use when:** A fast-paced product demo that shows specific features in sequence. Person is on screen demonstrating the product — tapping, swiping, pressing, unboxing, comparing. Think: app tour, gadget hands-on, before/after reveal.

**Read first:** [seedance.md](seedance.md) for platform rules and the adaptation checklist.

---

## What defines this style

- **Fast cuts:** Each feature gets 2–4 seconds — in and out
- **On-screen text:** Every feature beat has a label or callout that snaps on
- **Hands are the hero:** The product is experienced through what someone does with it
- **Energy escalates:** Each beat is slightly more impressive than the last
- **Verdict lands hard:** The final beat is the payoff — a reaction, a result, a "wow" moment

---

## The structure

```
 1. FORMAT HEADER    — duration, style descriptor, energy level
 2. TALENT           — appearance, energy, hand detail
 3. FEATURE SEQUENCE — 4–5 beats, each showing one feature
 4. TEXT CALLOUTS    — label for each feature beat
 5. PACING           — cut timing, escalation pattern
 6. VISUAL LANGUAGE  — lighting, color, camera
```

---

## Layer-by-layer formula

### Layer 1: Format header

```
15 seconds {{FORMAT_TYPE}} video, {{ENERGY}}, {{CAMERA_STYLE}}.
```

| Variable | Options |
|----------|---------|
| `FORMAT_TYPE` | product feature walkthrough, app demo, hands-on review, quick feature tour |
| `ENERGY` | fast-paced and punchy, energetic and direct, clean and confident |
| `CAMERA_STYLE` | dynamic handheld close-ups, crisp product-focused cuts, tight demo shots |

---

### Layer 2: Talent

```
A {{AGE_RANGE}} {{GENDER}} with {{APPEARANCE}}, {{SKIN_REALISM}},
wearing {{OUTFIT}}. Hands are {{HAND_DETAIL}} — {{NAIL_CONDITION}}.
```

**Skin realism (3–4 cues — always include):**
- `visible pores, slight unevenness in skin tone`
- `minor undereye shadows, hint of shine from natural oils`
- `the kind of skin you see on a real person, not retouched`

**Hand detail (important — hands are on screen constantly):**
- `clean, natural nails, no polish` / `subtle nude manicure`
- `hands look worked-in and real, not model-perfect`
- `slight skin texture visible on knuckles`

---

### Layer 3: Feature sequence

4–5 beats, each showing exactly one feature. Each beat: **action + result + label**.

```
[00:00] Hook — {{HOOK_ACTION}}. Text snaps on: "{{HOOK_LABEL}}"
[00:03] Feature 1 — {{FEATURE_1_ACTION}}. Text: "{{FEATURE_1_LABEL}}"
[00:06] Feature 2 — {{FEATURE_2_ACTION}}. Text: "{{FEATURE_2_LABEL}}"
[00:10] Feature 3 — {{FEATURE_3_ACTION}}. Text: "{{FEATURE_3_LABEL}}"
[00:13] Payoff — {{PAYOFF_MOMENT}}. Text: "{{CTA_OR_VERDICT}}"
```

**Feature beat bank:**

| Feature type | Action | Example label |
|-------------|--------|---------------|
| **Speed / performance** | Side-by-side comparison, timer visible | "3x faster." |
| **Texture / feel** | Fingers pressing, spreading, rubbing | "Melts in instantly." |
| **Capacity / size** | Pouring, filling, stacking | "Fits your whole routine." |
| **Ease of use** | One-hand operation, quick snap | "One click. Done." |
| **Before / after** | Split frame or quick pan reveal | "Day 1 vs Week 4." |
| **Social proof** | Number appearing on screen | "127,000 reviews." |

---

### Layer 4: Text callouts

```
Each feature beat has a text label that {{TEXT_BEHAVIOR}}.
Font is {{FONT_STYLE}}, color {{TEXT_COLOR}}, positioned {{PLACEMENT}}.
```

**Text behavior options:**
- `snaps on at the start of the beat, holds for 2 seconds` — punchy, ad-native
- `fades in over 0.5 seconds as the action completes` — smoother
- `slides in from below as the feature is demonstrated` — dynamic

**Font + placement:**
- Bold sans-serif, white text, centered below the action area — default
- Black text on white pill/badge — clean DTC look
- Uppercase with tracking — high-energy, sports/gadget feel

---

### Layer 5: Pacing

```
The edit is fast and rhythmic — each beat is {{BEAT_LENGTH}}.
Cuts happen on {{CUT_TRIGGER}}. The energy {{ESCALATION_PATTERN}}.
```

**Beat length:** `2–3 seconds per feature` (tight), `3–4 seconds per feature` (measured)

**Cut trigger:** `action completion` (natural) / `on the beat` (music-synced feel) / `at peak of the motion`

**Escalation:** `starts grounded and builds to the payoff beat` / `consistent energy throughout` / `slows down for the final verdict`

---

### Layer 6: Visual language

```
The lighting is {{LIGHTING}}. The background is {{BACKGROUND}}.
Color palette: {{PALETTE}}. Camera is {{CAMERA_DETAIL}}.
```

**Lighting options:** "clean studio-adjacent overhead," "natural window light," "ring-lit product close-ups," "warm neutral ambient"

**Background options:** "solid white or off-white," "neutral countertop," "product's natural use environment," "minimal desk or bathroom"

## Example: Skincare roller feature walkthrough

```
15 seconds product feature walkthrough video, fast-paced and punchy,
tight demo shots with clean product close-ups.

A woman in her early 30s with warm blonde hair down, medium skin with
visible pores, slight unevenness in skin tone, minor undereye shadows,
wearing a cream ribbed tank. Hands have clean, natural nails — slight
skin texture on knuckles, real and unhurried.

[00:00] She lifts the @(img1) (rose quartz face roller, silver handle)
from a white ceramic tray, holds it toward camera. Text snaps on:
"Meet your 60-second ritual."

[00:03] Close-up: she rolls it slowly across her cheekbone, skin
visibly cooling. Text: "Depuffs instantly."

[00:07] She tilts the roller to the smaller end, presses under her
eye, drags gently outward. Text: "Two sides. One tool."

[00:10] Wide shot — she points at her reflection in a round mirror,
laughs lightly at the visible difference. Text: "Less puff. More glow."

[00:13] Product centered on the ceramic tray, hands set it down. Text:
"Start tonight." Camera slowly pushes in.

The product appears identical in every beat — visually unchanged.
The edit is fast and rhythmic, each beat 2–4 seconds, cuts on action
completion. Text callouts are bold white sans-serif, centered below
the action. The lighting is soft natural window light from the left.
Color palette: cream, white, warm blush. No voiceover — text and
motion carry the story. No subtitles.
```

## Adaptation checklist

- [ ] 15 seconds, 4–5 beats with timestamps
- [ ] Talent described with skin realism cues + hand detail
- [ ] Each beat shows exactly one feature: action + result
- [ ] Every beat has a text callout — label + behavior specified
- [ ] Text font/style described (bold, color, placement)
- [ ] Pacing described (beat length + cut trigger)
- [ ] Product referenced as unchanged across all cuts
- [ ] Escalation pattern described (builds to payoff)
- [ ] Word count 100–260 words
- [ ] No forbidden words
